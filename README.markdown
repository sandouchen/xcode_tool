#### 更新方法

git add . (添加更新)

git status (查看状态)

git commit -m "" (创建更新提示)

git push origin master (上传到github工程目录下)

git pull origin master(更新)

#### PUSH失败做法
git pull --rebase origin master

git push -u origin master
